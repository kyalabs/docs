Teams run the rig on a cadence, not once.

A single study is a snapshot. The value compounds when the same mission runs on the same panel version across time, or against the same competitor set on a schedule. Then you have a comparable trendline, not a series of one-offs.

## The four common cadences

<CardGroup cols={2}>
  <Card title="Quarterly baseline" icon="clock">
    **The trendline.** Same mission, same panel version, every quarter: a durable read on how agents see your category, comparable quarter over quarter.

    _Best for:_ a trend over time, or trend into other commercial metrics.
  </Card>

  <Card title="Competitive watch" icon="radar">
    **Where you land vs. named rivals.** A standing read on which brands surface in the agent's answer for your category question, refreshed on a cadence you set.

    _Best for:_ category leaders who need a named-comparison view.
  </Card>

  <Card title="Pre / post-launch" icon="split">
    **Did the change move the read?** A controlled before-and-after on the same instrument. A site change, a campaign, a new product page: did it move how agents shop you, or not?

    _Best for:_ attribution on changes you've shipped.
  </Card>

  <Card title="Reach audit" icon="circle-half-stroke">
    **Which surfaces agents can reach.** A focused study on access posture: which of your surfaces agents reached, which turned them away, reported per finding as **reached** or **access-limited**.

    _Best for:_ surfaces that turned agents away when they should be open.
  </Card>
</CardGroup>

## Picking a cadence

The cadence determines what kind of question you can answer:

| Question you're asking | Cadence to run |
| --- | --- |
| _"Where do we stand today?"_ | One study, any posture: a snapshot |
| _"Are we moving over time?"_ | Quarterly baseline |
| _"Did that change work?"_ | Pre / post-launch |
| _"Where do we lose to whom?"_ | Competitive watch |
| _"Which surfaces are turning agents away?"_ | Reach audit |

<Note>
  Cadence is set with you during baseline. Most engagements land on a quarterly baseline plus one or two of the others. See [How it's offered → engagement lifecycle](/offered/engagement-lifecycle).
</Note>

## What makes a cadence comparable

A trendline only holds if the instrument doesn't drift, which is why the panel is versioned and bridge runs keep the trendline intact when a provider retires a model. See [the Panel as an instrument](/how-it-works/panel-as-an-instrument).
